工具指南

Spec Kit 是這套工作流的核心引擎。透過標準化的指令，讓你的自然語言需求精準轉化為程式碼。

安裝與初始化

1. 使用 uv 安裝 (推薦)

uv tool install specify-cli --from git+https://github.com/github/spec-kit.git

2. 在專案目錄初始化

specify init . --ai claude --here

Slash 指令對照表

指令 用途說明
/speckit.constitution建立專案憲法，定義最高指導原則與技術規範。
/speckit.specify描述功能需求與使用者故事 (User Stories)。
/speckit.clarify必做釐清模糊需求，確保 AI 完全理解規格。
/speckit.checklist產生驗收標準與測試清單。
/speckit.plan擬定技術實作藍圖與檔案結構。
/speckit.tasks將計畫拆解為可執行的細項任務 (TODOs)。
/speckit.implementAI 自動撰寫程式碼並完成任務。
